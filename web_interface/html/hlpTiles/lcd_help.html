LCD Popup Help Document
In addition to being used to display messages on the LCD of the mechbot this control can also be used to send commands to the system. Care should be taken when sending commands as there is no conformation or way to reverse a command once sent.

Command Function
REBOOT Restart the mechbot
REQUEST IP Display the mechbot's IP address on the LCD
RESET ROSSERIAL Restart the connection between the microcontroller and the main system
SHUTDOWN Shutdown the mechbot
